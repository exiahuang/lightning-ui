Introduce

Lightning-UI (LUI) is a Sleek, intuitive, and powerful front-end framework for faster and easier web development. It is assembled in my spare time. It is base on Salesforce Lightning Design System and Bootstrap. If you get the help from Lightning-UI, you can donate to Lightning-UI. With your help, I believe that I will let it be better.

The code is base on:

  • The Salesforce Lightning Design System
  • Bootstrap 3.3
  • Lightning-Table is Base on Bootstrap-Table
  • Simple javascript libary, Use grunt to build src, use jekyll
  • Document is base on jekyll
